# Minesweeper-pygame
pygame 踩地雷

# 遊戲玩法

在初始畫面中，你可以在主畫面點擊難度按鈕以選擇以下三種難度：

簡單：9*9 大小，10顆地雷

中等：16*16 大小，40顆地雷

困難：30*16 大小，99顆地雷

其餘規則請參考以下連結：

https://zh.wikipedia.org/zh-tw/踩地雷#玩法

## 操作說明

滑鼠左鍵：點開格子

滑鼠右鍵：標示地雷區域

遊戲結束後點擊畫面任一處：回到主畫面

（注意：在遊戲開始後，用滑鼠點到任何一個格子會開始計時）
